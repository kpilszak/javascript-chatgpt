## 🧠 General Info
This project is a **ChatGPT clone** built entirely with **vanilla JavaScript**.  
Its main purpose is to connect to the **OpenAI API** and fetch completions using the **GPT-5** model.

---

## 🛠️ Technologies
This project was created with:
- **JavaScript**
- **HTML5**
- **CSS3**
